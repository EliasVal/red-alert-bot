# Privacy Policy

This application only stores Discord guild IDs and announcement channel IDs required for operation.

- Only Discord guild ID and announcement channel ID are stored.
- No other user information is collected.
- Stored IDs are not shared, analyzed, or used for tracking.
- No third-party services are used to gather personal information.

Because only minimal configuration IDs are stored and no personal data is retained, there is nothing additional to delete or export.

If you have privacy concerns, you can use the app knowing it does not collect or share personal information beyond the necessary Discord identifiers.
